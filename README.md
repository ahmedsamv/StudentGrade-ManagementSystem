# StudentGrade-ManagementSystem
A CLI menu-based Bash application that manages students, subjects, and grades.

# Features 
- Students
  - Add Student
  - List Students
  - Update Student
  - Delete Student
 
- Subjects
  - Add Subject
  - List Subjects
  - Update Subject
  - Delete Subject

- Grades
  - Assign Grade
  - Update Grade
  - Delete Grade
  - View Grades by Student
  - View Grades by Subject

- Reports
  - Student GPA
  - Subject Statistics
  - Top Students
  - Failing Students
  - Full Grade Matrix
 
# How to Run
chmod +x sgms.sh
./sgms.sh
